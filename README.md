# friendsZone 

## LOCAL SETUP
1. install dependencies
```
$ sudo npm i -g react-native-cli
$ cd <project_dir>
$ npm install
```
3. Open a terminal. 

Run app on ios simulator
```
$ react-native run-ios
```
Run app on android simulator
```
$ react-native run-android
```

#### Reference
_https://reactnative.dev/docs/environment-setup_
