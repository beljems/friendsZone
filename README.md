# friendsZone 

## LOCAL SETUP
1. install dependencies
```
$ cd <project_dir>
$ npm install
```
3. Open a terminal. 

Run app on ios simulator
```
$ npx react-native run-ios
```
Run app on android simulator
```
$ npx react-native run-android
```

#### Reference
_https://reactnative.dev/docs/environment-setup_
